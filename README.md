# Bullet Journal Calendar

Персональний адаптивний календар-щоденник на Next.js і TypeScript.

## Запуск

```bash
npm install
npm run dev
```

Застосунок зберігає всі дані локально в браузері через `localStorage`: записи днів, калорії, кольори, стікери, позиції фото, шрифти й тему для кожного місяця окремо.
